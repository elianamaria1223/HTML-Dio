Sobre a hierarquia das tags h1 até h6

Exemplos:

Esta é a tag H1

Esta é a tag H2

Esta é a tag H3

Esta é a tag H4

Esta é a tag H5
E por fim, este á o tamanho da tag H6

Essas tags h1 até h6 srvem para nomear o conteúdo de textos que serão apresentados na sequência, geralmente de acordo com a relevância que se pretende dar ao título do assunto a ser descrito

Sobre tag p.

Essa tag serve para armazenar o texto ou o contéudo das frases, a letra p significa paragraph, ou seja, o parágrafo.

Sobre a tag mark.

Neste parágrafo vou destacar a tag mark, que serve como uma caneta amarela, ou mais conhecido como marcador de texto, que serve exatamente para chamar a atenção como no exemplo da palavra destacada.

Sobre a tag small

Essa tag smallserve para diminuir a letra em relação ao tamanho da letra da frase, como feito neste exemplo. Muito usada em rodapés.

Sobre a tag strong

Esta tag strong é utilizada não apenas para estilizar em negrito a palavra, mas acima de tudo por sua importância semântica dentro do texto.

Para fazer uma lista ordenada, na qual é fundamental seguir a ordem dos passos dados, utiliza a tag ol.Dentro da lista utilize a tag li, que significa "list item".Como exemplo na lista abaixo:

  1. Separe os ingredientes do bolo de chocolate
  2. Bater tudo no liquidificador
  3. Assar em forma untada po 45 min
  4. O bolo está pronto

Porém se você precisar fazer uam lista de compras do supermercado e não precisa seguir uma ordem, pode utililizar uma ul, que significa unordered list. Para acrescentar os itens a serem comprados dentro desta lista utilize as mesmas tag li.Veja o exemplo:

  • maça
  • banana
  • leite
  • ovos
